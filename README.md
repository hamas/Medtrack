# Medtrack — Professional Medicine Tracking

**Lead Developer:** Hamas

## Vision
Medtrack is a "Google Task-style" medicine tracker designed from the ground up for medical-grade precision. It pairs an intuitive, clean interface with robust, reliable tracking, ensuring you never miss a dose while providing deep insights into your adherence history.

## Core Features
- **Complex Scheduling Algorithms:** Native support for daily, bi-weekly, monthly, quarterly, and fully customized interval tracking.
- **Contextual Medical Delivery:** Log precise meal context (before/with/after meals) and dynamic delivery methods (e.g., water, injection, inhaler).
- **Cross-Device Sync:** Cloud-powered state propagation ensuring your schedule and check-offs are always up to date securely.
- **Advanced History Export:** Generate clinical-grade PDF adherence reports directly for doctors.
- **Biometric Security:** Enterprise-level access gates ensuring only authorized individuals can view sensitive history.

## Tech Stack
- **Framework:** Flutter (100% Dart Philosophy)
- **Backend Sync:** Firebase (Auth, Firestore)
- **State Management:** Riverpod 
- **Security:** Local Auth, Flutter Secure Storage
- **Routing:** GoRouter

## Architecture
Medtrack adheres strictly to the **Feature-First Clean Architecture**, separating the application into decoupled, highly testable domains (`core`, `features`, `services`) with unidirectional data flows.
